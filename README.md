  # 🔭얼렁뚱땅 CV 논문 CODE 훑어보기 
  </div>
  
---

코드 뜯기 및 논문 관련 리뷰는 블로그에 업로드 할 예정이나 가능하다면 ipynb파일로 정리해서 올릴 예정입니다. 

---

- **PyTorch로 진행하니다.**
- 기간: 총 6주 과정(3월 21일 ~ 4월 24일)
- 시간: 매주 금요일 20시 30분 ~ 23시 30분
- 일정
  - 오티 및 전체적인 스터디 과정 회의
  - Faster R-CNN / YOLOv1
  - FPN / YOLO:9000(v2)
  - U-net / YOLOv3
  - Mask R-CNN / YOLOv4
  - ViY / YOLOv5

---

📌**진행 방식**
- 본 스터디는 논문 리뷰가 아닌 논문 코드 리뷰를 진행하니다(논문을 본 사람만이 참여가 가능한 것은 아닙니다).
- 스터디는 2 Session으로 진행할 예정입니다(약 3시간정도 예상).
- 주당 2편의 논문 중 한 편을 택하여 학습을 진행하면 됩니다(하나의 논문으로 몰리는 것을 생각해, 추각적인 방법을 통해 반반 나눌 예정입니다).
- 각 Session마다 스터디를 이끌어 가실 "리더"를 뽑아 진행될 예정입니다.
